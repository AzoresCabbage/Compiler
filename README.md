Compiler
========

基于web端的编译器词法分析和语法分析


实现了词法分析和LR1语法分析。
code.txt是待分析的代码；
keyword.txt是语言的关键词；
lex.cpp是词法分析代码；
lex_analyze.txt是词法分析的结果；
lex_grammar.txt是词法的文法表示；
LR1_table.txt是LR1语法分析产生的LR1表；
main.cpp是主程序调用文件；
project_set.txt是LR1语法分析生成的项目集规范族；
syntax.cpp是语法分析的代码；
syntax_analyze.txt是语法分析的结果；
syntax_grammar.txt是语法的文法。
其余文件为.net网站文件。
代码高亮使用了codemirror插件。
